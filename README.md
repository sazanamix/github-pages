# さざぶろぐ

個人的なメモ、備忘録、実験サイトです。

公開URL: https://sazanamix.github.io/github-pages/

## 構成

このリポジトリは GitHub Pages でホスティングしています。直下の `index.html` がトップページで、各サブフォルダが個別のランディングページです。新しいページを追加する場合は、フォルダを作って `index.html` を置き、トップページ (`index.html`) にリンクを追加してください。

## コミットメッセージ規約

`<type>: <理由>ため、<変更内容>` の形式で記載しています(例: `feat: 検索性能を改善するため、インデックスを追加`)。
