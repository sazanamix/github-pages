補足メモ

  • UWSCR は「正式な後継」ではありません。UWSC 作者とは無関係に、動作を模倣して一から開発された互換ツールです。構文は寄せていますが完全一致は保証されず、資産の移行時は動作検証が必要です。
  • UWSC 本体は 2017 年で更新停止。本体自体は今も動きますが、CDO メール送信など周辺の前提が古く、現行環境で動かない例が増えています。急いで乗り換える必要がなければ使い続けても構いませんが、ブラウザ操作など不都合が出たら移行の合図です。
  • 配布のしやすさは実務で効きます。UWSCR / AHK / Python はスクリプト（テキスト）配布が容易。Power Automate はフローの export/import やアカウント連携に制約があり、配布は相対的に手間です。
  • この表は 2026 年 7 月時点の一般的な評価です。最終判断の前に、実際の対象業務（ブラウザ操作の有無、Excel 依存度、社内ガバナンス要件）で PoC することをおすすめします。
